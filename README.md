# Friends of Chitambo

## What
A static site with information about registered Scottish charity, Friends of Chitambo (SCO44337)

## Development

Dev Build
---------
```bash
$ npm install
$ npm start
```

Now point your browser to http://localhost:8000/.

Production Build
----------------
```bash
$ npm run build
```

Deploy to GitHub Pages
----------------
```bash
$ npm run deploy
```
